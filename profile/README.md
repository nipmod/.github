# Nipmod

The package intelligence layer for AI agents.

Nipmod helps agents search public package sources, inspect trust signals and request safe install plans before code enters a workspace.

The hosted API is read-only for user workspaces. It returns package context, warnings and install-plan data; execution remains with the user's local host or agent runtime.

## Current Focus

- API-first package intelligence for agents.
- Source resolvers for npm, PyPI, GitHub, Hugging Face and MCP.
- Trust signals, warnings and safe install plans.
- Durable package intelligence archive.
- Open source CLI and MCP workflows.

Nipmod does not replace npm, PyPI, GitHub, Hugging Face or MCP. Original package owners keep ownership. Nipmod adds source context, trust checks and agent-readable package intelligence above those ecosystems.
