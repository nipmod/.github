# Nipmod

One package API for agents.

Nipmod helps agents search package sources, inspect trust signals and request safe install plans before code enters a workspace.

## Current Focus

- Hosted package API for agents.
- Source resolvers for public package ecosystems.
- Trust signals and safe install plans.
- Durable package intelligence archive.
- Open source CLI and MCP workflows.

External package owners keep ownership. Nipmod adds source context, trust checks and agent-readable package intelligence.
